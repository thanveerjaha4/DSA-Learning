# What is DSA?

DSA means:
--Data Structures + Algorithms

## Data Structure = HOW you store data

Example:
- List of names → Array
- Stack of plates → Stack
- Friends network → Graph

" It is just a way to keep data organized"

##  Algorithm = HOW you solve a problem

Example:
- Find biggest number
- Search a name
- Sort numbers

- It is step-by-step solving method.


##  Simple idea:

DSA = Store data + Solve problem in best way

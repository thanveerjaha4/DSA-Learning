# Hashing

This folder contains notes and Python solutions related to the Hashing technique.

## Topics Covered

- What is Hashing?
- HashMap vs HashSet
- Frequency Count

## Problems Solved

- Contains Duplicate (LeetCode 217)
- Two Sum (LeetCode 1)
- Valid Anagram (LeetCode 242)
- Intersection of Two Arrays (LeetCode 349)
- First Unique Character in a String (LeetCode 387)

## Goal

Learn how to use HashMaps and HashSets to solve array and string problems efficiently by reducing time complexity from O(n²) to O(n) in many cases.

# Binary Search

This folder contains notes and Python solutions related to the Binary Search technique.

## Topics Covered

- Binary Search Basics
- Iterative Binary Search
- Recursive Binary Search

## Problems Solved

- Binary Search (LeetCode 704)
- Search Insert Position (LeetCode 35)
- Guess Number Higher or Lower (LeetCode 374)
- First Bad Version (LeetCode 278)
- Valid Perfect Square (LeetCode 367)

## Goal

Learn how Binary Search efficiently finds an element in a sorted array by repeatedly dividing the search space into halves, reducing the time complexity from **O(n)** to **O(log n)**.

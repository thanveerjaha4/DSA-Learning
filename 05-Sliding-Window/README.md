# Sliding Window

This folder contains notes and Python solutions related to the Sliding Window technique.

## Topics Covered

- Sliding Window Basics
- Fixed Size Window
- Variable Size Window

## Problems Solved

- Maximum Average Subarray I (LeetCode 643)
- Contains Duplicate II (LeetCode 219)
- Maximum Number of Vowels in a Substring of Given Length (LeetCode 1456)
- Permutation in String (LeetCode 567)
- Longest Substring Without Repeating Characters (LeetCode 3)

## Goal

Learn how to efficiently solve subarray and substring problems by maintaining a moving window instead of recalculating values repeatedly.

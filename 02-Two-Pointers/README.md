# Two Pointers

The Two Pointer technique is a common DSA pattern used to solve array and string problems efficiently. Instead of using nested loops, we use two indices (pointers) to traverse the data, reducing the time complexity in many problems.

##  Why Learn Two Pointers?

The Two Pointer technique is widely used in coding interviews because it helps solve many problems in **O(n)** time while using **O(1)** extra space. It is commonly applied to arrays, strings, linked lists, and sliding window problems.

---

##  Summary

The Two Pointer technique is one of the most important DSA patterns. Mastering it will help you solve many interview questions efficiently and build a strong foundation for advanced topics like Sliding Window and Linked Lists.

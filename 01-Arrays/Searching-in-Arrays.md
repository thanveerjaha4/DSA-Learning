# Searching in Arrays

## What is Searching?

Searching means finding a particular element in an array.

---

## Why is Searching Important?

- Find whether an element exists.
- Find the position of an element.
- Used in many DSA problems.

---

## Types of Searching

### 1. Linear Search

Checks each element one by one.

Example:

Array:

[5, 8, 2, 9, 1]

Find: 9

Check:

5 

8 

2 

9 

Found at index 3.

Time Complexity: O(n)

---

### 2. Binary Search

Works only on a sorted array.

Instead of checking every element, it repeatedly checks the middle element.

Time Complexity: O(log n)

---

## Key Points

- Linear Search works on both sorted and unsorted arrays.
- Binary Search is faster but requires the array to be sorted.

---

## Summary

Searching is one of the most common operations in arrays.

The two most popular searching techniques are:

- Linear Search
- Binary Search

# Array Basics

## What is an Array?

An array is a collection of elements stored together in a single variable.

Example:

[10, 20, 30, 40, 50]

Here,
- Index 0 → 10
- Index 1 → 20
- Index 2 → 30
- Index 3 → 40
- Index 4 → 50

---

## Why do we use Arrays?

- To store multiple values together.
- To access elements quickly using an index.
- Easy to traverse using loops.

---

## Real-Life Examples

- Student marks
- Shopping list
- Mobile contacts
- Music playlist

---

- Access
